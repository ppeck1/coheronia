## How to run right now

1. Open `/g1v4/` folder in Godot 4.x (4.3+ recommended).
2. Let the editor import `terrain_placeholder.png` (happens automatically).
3. Press **Play** (F5). Godot will run `scenes/main/Main.tscn`.
4. You should see:
   - Player spawning on the ground platform
   - Procedurally generated terrain tiles rendered on screen
   - C/L/R bars in the HUD updating every 5 seconds
5. **Hold left mouse button** near a tile to mine it (0.6s per tile).
   - Check the Output panel — you'll see `Inventory: {...}` printed on each mine.
6. Press **T** to toggle the Town Hall stub panel.

## Known limitations (deferred to later phases)

- **No tile placement yet** — inventory fills but you can't place blocks back. (Phase 2: Hotbar UI + right-click place)
- **Mining progress bar** is a Node2D ColorRect in world space, not screen UI — it may look small. (Phase 2: proper screen-space overlay)
- **No combat damage** — player and enemies can collide but deal no damage. (Phase 3)
- **RaidDirector is disabled** (`enabled = false`) — no raids yet. (Phase 6)
- **The StaticBody2D ground platform remains active** alongside TileMap collision. This is intentional for now — the TileMap tiles generated by WorldGen have physics collision shapes baked in, so once you dig into the terrain, the tile-based collision takes over. The flat platform just ensures you land somewhere on first spawn. Set `$Ground.visible = false` in the World scene once your tunnel network is deep enough.
- **C/L/R observables are still all stubs** in `settlement.gd::gather_observables()` (food_days hardcoded to 2.0, etc.). These become real in Phase 5.

## Architecture reminder

```
Main.tscn
  └── GameRoot (game_root.gd)
        ├── World (world.gd)
        │     ├── TileMapForeground  ← terrain / foreground tiles
        │     ├── TileMapBackground  ← wall tiles (background layer)
        │     ├── Ground (StaticBody2D) ← temporary fallback floor
        │     ├── PlayerSpawn (Marker2D)
        │     ├── TownHallMarker (Marker2D)
        │     └── Player (player.gd) ← spawned at runtime
        ├── SettlementRoot (settlement.gd)
        │     ├── TownHall (Area2D)
        │     ├── Followers (Node2D)
        │     ├── Buildings (Node2D)
        │     └── SettlementTick (settlement_tick.gd) → CoherenceModel
        ├── HUD (hud.gd, CanvasLayer)
        └── RaidDirector (raid_director.gd) [disabled]
```
