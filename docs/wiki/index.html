Welcome

Built Like A Wiki Main Page, Grounded Like A Registry

The job of this page is not just to say what exists. It is here to show what exists, what is actually player-facing, what is internal, what is planned-only data, and what visual surfaces are live, deferred, or deliberately placeholder. The markdown source remains authoritative; this wrapper is the clearer, more welcoming browsing layer.

This wiki is generated from current Coheronia repo data and includes live, internal, and planned-data references. Planned-data pages are not live gameplay unless marked as live.

How To Use This

Start here for orientation, then jump into the detailed docs for exact tables. Use the front page when you need to answer what is live, what is planned, what is missing an art pass, or what is present in data but not meaningful in gameplay yet.

What This Does Not Do

This page does not invent mechanics or silently redesign systems. Proposed sinks and future hooks are called out as proposals only, and dead or internal tokens stay explicitly labeled so they do not get mistaken for implemented player items.
